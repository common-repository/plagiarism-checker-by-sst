== Installation ==

You can use the built in installer and upgrading feature, or you can install the plugin manually.

1. You can use the built-in installer and upgrading feature, or you can install the plugin manually.
1. You can either use the automatic plugin installer or your FTP program to upload it to your wp-content/plugins directory the top-level folder. Don’t just upload all the PHP files and put them in `/wp-content/plugins/`.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Visit your Options Page to check plugin status.
1. Configure any options as desired, and then enable the plugin.
1. That’s it!


After Installation, create an account at [SmallSEOTools.com](https://smallseotools.com/) to get API key for your WordPress plugin and put that API key on the plugin settings page.
